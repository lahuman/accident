# Car Accident 프로젝트

## 개요
Car Accident 프로젝트는 자동차 사고 데이터를 처리하고 분석하기 위한 시스템입니다.

## 사용 기술
- **TypeScript:** 주요 프로그래밍 언어
- **NestJS:** 서버사이드 애플리케이션 프레임워크
- **JavaScript:** 보조 스크립팅 언어
- **ESLint & Prettier:** 코드 품질 및 스타일 관리 도구
- **PNPM:** 패키지 매니저

## 실행 방식
1. 프로젝트를 클론합니다.
2. 종속성을 설치합니다:
    ```bash
    pnpm install
    ```
3. 서버를 시작합니다:
    ```bash
    pnpm start
    ```
